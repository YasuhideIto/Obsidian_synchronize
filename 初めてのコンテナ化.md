
### [[コンテナ化]]できるようになったプロジェクト

### [[docker]]でコンテナ化して起動
手順

- プロジェクトのパスに移動

>[!caution] すでにpull してきたプロジェクトの場合
 git remote -v でURLをローカルがもってるか確認
ex)     
origin https://github.com (fetch)
origin https://github.com (push)
>

- terminalでgit --version確認
- git add . （すでにURLをもってるのでgit initは不要）
- git commit -m "任意のコメント"
- git push origin main
