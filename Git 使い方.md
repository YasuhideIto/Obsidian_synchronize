
### 別PCでリモートのファイル情報とブランチ情報の反映


💻 別のPCで新しいブランチを反映させる手順

新しいブランチを別のPCに同期させるには、**「本棚に新しい本を追加する（fetch）」**、そして**「その本を開いて作業を始める（checkout）」**という2つのステップが必要です。

1. サーバーから最新のブランチ情報を取ってくる

まず、新しいブランチが作られたという「通知」を別PCに送ります。

bash

```
git fetch origin
```

コードは注意してご使用ください。

- ※ `git pull` でも情報は届きますが、`fetch` の方が「ブランチの情報だけ」を安全にスッキリ取得できるため、Gitではこちらが推奨されます。

2. 新しいブランチに切り替える（ジャンプする）

届いた新しいブランチへ、自分の作業画面を切り替えます。

bash

```
git checkout <新しいブランチ名>
```

コードは注意してご使用ください。

- ※ 最近のGitを使っている場合は `git switch <新しいブランチ名>` でも同じです。
- ※ このコマンドを打つだけで、自動的にリモートと同じ名前のブランチが手元に作られ、連動します。

3. 確認する

無事に切り替わったか、一覧を表示して確認します。

bash

```
git branch
```

コードは注意してご使用ください。

- 現在いるブランチ名の横に **`*`（アスタリスク）** が付いていれば、同期と切り替えはすべて完了です！
